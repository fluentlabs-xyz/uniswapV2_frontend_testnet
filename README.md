## Run frontend locally for testing

⚠️ Node.js version v16.14.2 is recommended to avoid errors running the website locally. ⚠️
```shell
npm install http-server
```
then
```shell
npx http-server
```
or
```shell
http-server
```
Note: this website example uses Vanilla Javascript.
